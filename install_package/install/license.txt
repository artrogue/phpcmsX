Phpcms X
基于phpcms v9.6.3 UTF8 完善和优化
保留所有官方原版程序功能,保持官方原程序前后台界面,方便二次开发;
主要完善和优化项如下:

支持后台一键更新(限使用Phpcms X程序)
支持HTTPS环境
支持php7+ (最高支持到 php7.3.8 )
支持MySQL8+ (最高支持到 MySQL 10.4.13-MariaDB)
增加安装时自定义后台管理登录地址
增加远程更新修复等相关信息获取显示
增加讯飞,百度关键词提取接口(替代PHPCMS官方失效的接口)
后台附件上传修改为H5上传
会员头像上传修改为H5上传
去除了已被废弃的视频模块和视频模型
修复安装时DNS解析错误提示
修复phpsso和Ucenter在https下通信相关功能
修复phpsso中一处bug
修复已知安全漏洞
修复其他已知bug
更多功能及模块整合不断更新中(预计3.0版更新上线内置模块商店)
